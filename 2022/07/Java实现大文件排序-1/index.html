Java实现大文件排序-1

编程 Java
发布日期:   2022-07-22
更新日期:   2022-07-22
文章字数:   469
阅读次数:  

大文件排序 （入口，常量类与初始化类）

入口 Entry类

  1. 设置错误日志
  2. 输入主机编号
  3. 开始初始化
  4. 启动流程
  5. 流程结束与收尾

常量类

1. CONST类 简单常量

  • debug debug开关
  • optFiles 配置文件类 （“D:/Desktop/Test/opts.txt”）
  • ALPHA_NUM 字母数量 （26）
  • ALPHABET 字母表
  • WORD_LEN 单词长度 （16）
  • PAK_LEN 包长度 （10）
  • BYTE_CACHE 读取缓存 （16*1000）
  • PAK_CACHE 包缓存 （10*1000）
  • b2i() 字节转整形函数
  • ErrorLog 错误日志
    • getErrorLog()
    • setErrorLog()
    • closeErrorLog()

2. THR类 线程常量

  • ThreadPool 主线程池
  • ThrPool 副线程池
  • ExecWait() 等待空闲线程
  • PoolWait() 等待线程池执行完毕
  • ThreadPoolExecutorFactory 线程池工厂
  • THR_NUM 核心线程数 （20）
  • THR_MAX_NUM 最大线程数 （80）
  • B_MAX_NUM 线程阻塞队列大小 （1000）
  • THR_ALIVE_TIME 线程最大存活时间 （3s）

3. NET类 网络流常量

  • IP_idx 本主机编号 （0~7）
  • PORT 服务器端口 （4700）
  • IPs IP类
  • END 网络传输结束符（PAK_CACHE）
    • ENDFactory() 结束符工厂

4. FileSL类 文件处理类，封包对文件操作，简化代码

  • LoadFile() 读取文件，将文件中每一行存储于Array<String>
  • CharFile() 生成字母命名的txt文件
  • ResultFile() 生成结果文件
  • FileFactory() 文件工厂，判断文件（夹）,若文件（夹）不存在则创建文件（夹）

初始化类

动态确定配置文件，便于在不同配置环境下调试

1. Files （F开头为文件夹，f为文件）

  • FInput 输入 （“D:/Desktop/Test/FIn”）
  • FOutput 切分文件输出（/网络流输入） （“D:/Desktop/Test/FOut”）
  • FNet 网络流输出（/排序） （“D:/Desktop/Test/FNet”）
  • FResult 输出 （“D:/Desktop/Test/FResult”）
  • fIP IP地址 （“D:/Desktop/Test/IP.txt”）
  • fError 错误日志 （“D:/Desktop/Test/Errorlog.txt”）

2. IP

  • IP_num 主机数量 （8）
  • IP_address IP地址
  • IP_alpha IP对应字符串
  • IP_String() 根据主机数量与给定切分依据，获取IP对应字符串，便于在不同主机数量时分配字符
