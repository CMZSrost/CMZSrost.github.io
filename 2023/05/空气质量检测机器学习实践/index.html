背景引入

任务要求

空气质量预测（给定一段时间内的北京天气相关指数数据和北京PM2.5指数等，建立K近邻模型预测接下来一段时间内北京的PM2.5指数。）

数据说明

数据主要包括2010年1月1日至2014年12月31日间北京pm2.5指数以及相关天气指数数据。

数据分为训练数据和测试数据，分别保存在train.csv和test_noLabel.csv两个文件中。

其中训练数据主要包括35746条记录，13个字段，主要字段说明如下：

字段名 描述
ID 样本编号
date 观测数据发生的日期（年-月-日）
hour 观测数据发生的时间点（时）
Label 观测时间点对应的pm2.5指数（(ug/m^3)
DEWP 露点，空气中水气含量达到饱和的气温（℃）
TEMP 温度，观测时间点对应的温度（℃）
PRES 压强，观测时间点对应的压强（hPa）
Iws 累积风速，观测时间点对应的累积风速（m/s）
Is 累计降雪，到观测时间点为止累计降雪的时长（小时）
Ir 累计降雨，到观测时间点为止累计降雨的时长（小时）
cbwd_NE 观测时间点对应的风向为东北风(m/s)
cbwd_NW 观测时间点对应的风向为西北风(m/s)
cbwd_SE 观测时间点对应的风向为东南风(m/s)
cbwd_cv 观测时间点对应的风向为静风(m/s)

测试数据主要包括6011条记录，12个字段，测试数据的字段信息和训练数据相比，除了不包括Label字段以外其他完全相同。

排名说明

算法通过计算平均预测误差来衡量回归模型的优劣。平均预测误差越小，说明回归模型越好。

优化思路

题目限定模型为KNN模型，因此优化方向为特征工程与超参数优化。

特征工程

数据预处理

  • 将数据集中的时间序列数据进行处理，切分为年月日并转换为数值型数据
  • 删去ID列
  • 对数据集中的缺失值进行处理
  • 将数据集进行标准化处理

数据探索

对数据集进行探索，包括数据集的箱线图与数据集的主成分分析等。

从数据集中可以看出，DEWP\PRES具有趋势，TEMP具有日间周期性。因此可以将数据按小时进行分组，消除周期性与趋势影响。

特征选择

为每个特征设置权重，后续通过贝叶斯超参数优化算法，优化权重，得到最优权重。

超参数优化

首先对KNN模型的超参数进行优化，KNN模型的超参数包括n_neighbors、weights、p、leaf_size等。其中n_neighbors为KNN模型的邻居数量，weights为KNN模型的权重，p为KNN模型的距离度量方式，leaf_size为KNN模型的叶子节点数量。

使用KNN对处理过的训练集进行10折交叉验证，使用均方误差作为评价指标，对KNN模型的超参数k进行优化，结果如下，得到最优k为2。

k=1-50

k=50-200

参数优化算法有随机搜索、网格搜索、贝叶斯优化等。由于选择特征数量多，维度大，因此选择贝叶斯优化算法。

贝叶斯优化，即贝叶斯超参数优化算法，是一种基于贝叶斯定理的优化算法。通过目标函数的返回值来修正先验分布，从而得到最优参数。贝叶斯优化算法的优点是可以通过少量的迭代次数，得到较好的结果。

optuna是一个超参数优化框架，支持贝叶斯优化算法，支持多种优化算法，包括TPE、GP等。本次实验使用TPE算法。optuna使用简单，只需定义目标函数，即可进行优化。

# 定义目标函数
def optuna_objective(trial):
    n_neighbors = trial.suggest_int('n_neighbors', 2, 300, 1)
    # leaf_size = trial.suggest_int('leaf_size', 10, 30, 1)
    p = trial.suggest_int('p', 1, 5, 1)
    # weights = trial.suggest_categorical('weights', ['uniform', 'distance'])
    lens = x_train.shape[1]
    weights = [trial.suggest_float(name, 0, 1, step=0.01, log=False) for name in [f'w{i}' for i in range(1, lens + 1)]]
    model = KNeighborsClassifier(n_neighbors=n_neighbors, p=p)

    cv = KFold(n_splits=10, shuffle=True, random_state=7)
    validate_loss = cross_validate(model, weights * x_train, y_train, cv=cv, scoring='neg_mean_squared_error',
                                   n_jobs=12, error_score='raise')

    return -np.mean(validate_loss['test_score'])


def optimizer_optuna(n_trials, algo):
    if algo == 'TPE':
        algo = optuna.samplers.TPESampler(n_startup_trials=10, n_ei_candidates=24)
    elif algo == 'GP':
        from optuna.integration import SkoptSampler
        algo = SkoptSampler(skopt_kwargs={'base_estimator': 'GP', 'n_initial_points': 10, 'acq_func': 'EI'})
    study = optuna.create_study(sampler=algo  # 定义样本抽样的算法
                                , direction='minimize'  # 定义目标函数优化方向
                                )
    study.optimize(optuna_objective  # 目标函数
                   , n_trials=n_trials  # 设定最大迭代次数（包括最初观测值）
                   , show_progress_bar=True  # 展示进度条
                   )
    print(f'best parmas:{study.best_trial.params}')
    print(f'best score:{study.best_trial.values}')
    return study.best_trial.params, study.best_trial.values

实验结果

经过多次迭代，最终得到最优权重为[0.82, 0.4, 0, 0.75, 0.35, 0.2, 0.0, 0, 0.0, 0, 0.4, 1, 0.7, 0]，参数k=2，p=1。最终得分为3867.3745，排行第二，仅次于使用多决策树投票模型的第一名3551.0161。

    x_data.iloc[:, :] = scaler.fit_transform(x_data)

def plot_time(dat, feature):
    dates = dat['date'].astype(str).str.cat(dat['hour'].astype(str))
    for i in feature:
        plt.figure()
        plt.scatter(dates, dat[i].values, s=1)
        xx = list(set(dates))
        xx.sort()
        p = 12 * int(len(xx) / 1000)
        plt.xticks(xx[::p + 1], rotation=45)
        plt.savefig(f'time_png/time-{i}.png', dpi=600)

def create_data(inputPath, train=True):
    '''
    :param inputPath:数据集文件路径
    :return: 清洗后的数据集与数据结果
    '''
    y_test = None
    x: pd.DataFrame = pd.read_csv(inputPath)
    x.drop('ID', axis=1, inplace=True)

    feature = x.columns.values
    feaSets = dict(zip(feature, ['float' for i in range(len(feature))]))

    x.insert(11, 'month', pd.to_datetime(x['date']).dt.strftime('%m'))
    x.insert(11, 'day', pd.to_datetime(x['date']).dt.strftime('%d'))
    x['date'] = pd.to_datetime(x['date']).dt.strftime('%Y')
    feaSets['date'] = 'int'
    feaSets['month'] = 'int'
    feaSets['day'] = 'int'

    # if train:
    #     plot_time(x.sort_values(['date', 'hour']), feature[1:-7])
    #     plot_time(x.sort_values(['date', 'hour']), ['Label'])

    x = x.astype(feaSets).dropna(axis=0)
    if train:
        y_test = x.iloc[:, -1].astype(int)
        x = x.iloc[:, :-1]
    describe_data(inputPath, x.copy())
    if train:
        return x.values, y_test.values
    else:
        return x.values


def to_csv(OutputPath, InputPath, y_pred):
    id = pd.read_csv(InputPath).values[:, 0].flatten()
    data = np.array([id, y_pred.flatten()]).T
    output = pd.DataFrame(data, columns=['ID', 'Label'], index=None)
    output.to_csv(OutputPath, index=False)


def getsel(featureNum):
    fea = list(range(1, 2 ** featureNum))
    lens = list(range(1, featureNum + 1))
    res = []
    for f in fea:
        sel = []
        i = f
        cnt = 0
        while i > 0:
            if i % 2 == 1: sel.append((lens[cnt]))
            cnt += 1
            i = int(i / 2)
        sel.append(-1)
        res.append(sel)
    return res
